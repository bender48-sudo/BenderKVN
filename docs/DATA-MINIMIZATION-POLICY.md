# Политика минимизации данных (P3-RED-MIN-01)

Публичная выжимка для пользователей. Юридически полные тексты — по ссылкам **`TERMS_URL`** / **`PRIVACY_URL`** в боте (telegra.ph).

## Что собираем (минимум для сервиса)

| Категория | Примеры | Зачем |
|-----------|---------|--------|
| **Идентификатор Telegram** | numeric `telegram_id`, опционально `@username` | аккаунт, поддержка, начисление баланса |
| **Технические данные VPN** | UUID ключа, срок подписки, шаблон в панели | выдача и продление доступа |
| **Биллинг (агрегаты)** | баланс в ₽, сумма пополнений, id платежа у PSP | оплата и защита от двойного начисления |
| **Поддержка** | topic id, метки SLA | ответы в чате |

## Что **не** собираем

- Паспорт, адрес проживания, обязательный email или телефон для регистрации  
- Полные данные банковской карты (PAN/CVV) — обработка у **YooKassa** / **Telegram**  
- Историю сайтов и содержимое трафика VPN (провайдер не ведёт DPI-логи содержимого)  
- Точную геолокацию устройства  

## Разделение следов

- **Платёж** (ID у кассы, сумма) хранится в боте **отдельно** от технического **UUID** подписки в панели Remnawave.  
- Webhook PSP в БД бота сохраняется в **урезанном** виде (без карточных полей) — см. **`payload_redact.py`**.

## Сроки хранения (ориентиры)

| Данные | Срок |
|--------|------|
| Аккаунт и ключи | пока пользователь пользуется сервисом |
| Webhook idempotency (`webhook_deliveries`) | **90 дней**, затем purge (runbook) |
| Логи действий (`user_actions`) | до **12 мес.** |
