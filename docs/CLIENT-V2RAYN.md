# Windows: v2rayN + подписка BenderVPN

**Задача:** P1-PRO-CLIENT-V2RAYN-01 (Q052).

## Рекомендуемая версия

- **v2rayN** 7.x (VLESS/Reality поддерживается).
- Скачивание: [releases v2rayN](https://github.com/2dust/v2rayN/releases) — portable ZIP.

## Импорт подписки

1. Получите ссылку подписки в боте (**«Моя настройка»**) или на `https://k9x2m1.conntest.xyz:8443/setup/`.
2. v2rayN → **Подписки** → **Добавить** → вставьте URL (должен быть **`https://…:8443/api/sub/…`** после миграции edge).
3. **Обновить подписку** → выберите профиль **turbo** (NL/LV) для обычного сценария.
4. Системный прокси или **TUN** — по желанию; при проблемах начните с режима без TUN.

## Если не подключается

| Симптом | Действие |
|---------|----------|
| TLS / Reality error | Обновите подписку; проверьте, что URL на **:8443**, не **:2053** |
| Пустой список серверов | Проверьте User-Agent (должен быть клиент подписки); откройте URL в браузере — не должен отдавать HTML-ошибку |
| Работает Wi‑Fi, не LTE | Попробуйте другой tier в подписке; см. **`PRODUCT-TIER-PROFILES.md`** |
| Конфликт с Happ | Не запускайте два VPN одновременно |

## Happ vs v2rayN

- **Happ** — основной клиент в portal (проще).
- **v2rayN** — для пользователей Windows, которым нужен классический PC-клиент.

## Проверка (владелец / smoke)

- Импорт sub URL → connect → `https://ya.ru` открывается.
- Smoke: **`V2RAYN_CLIENT_OK`** — наличие этого файла + блока Windows в `web/portal/content/ru.json`.
