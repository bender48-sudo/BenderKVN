## Q057 — selfsteal / decoy (п.5) · решение **2026-05-18**

| | |
|--|--|
| **Решение** | **KEEP (упрощённый мониторинг)** — не выключать на проде без отдельного change window. |
| **Почему** | NL **:9443** decoy даёт легитимный TLS fingerprint; отключение без замены увеличивает «голый» probe на panel inbound. |
| **Хвост** | Сократить список SNI в `selfsteal-monitor.py` при следующем TLS-ревью (**P2-RED-TLS-01**); не расширять decoy без метрики. |
| **Smoke** | **`SELFSTEAL_REVIEW_OK`** §12 |
