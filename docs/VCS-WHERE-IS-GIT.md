# Где Git и как появится GitHub

Проверка каталога рабочей копии **`d:\\Va\\projects\\VPN`** (май 2026): директория **не** содержит **`.git`**, то есть **версионирования Git в этом дереве пока нет** — есть только файлы локально и код на VPS (прод по **`docs/DEPLOY.md`**). Отсюда **нет ни веток, ни удалённых репозиториев**.

Если репозиторий лежит в другой папке (другая копия диска, архив или облако) — нужно искать там признак **`/.git`** (скрытый каталог).

**Как завести с нуля:**

1. `git init` в корне того каталога, который считаем SoT-кодом для BenderVPN.  
2. `.gitignore` — как минимум `.secrets/`, `*.env`, крупные артефакты.  
3. Первые коммиты без секретов.  
4. На GitHub: **New repository** (пустой) → **`git remote add origin …`** → **`git push -u origin main`**.

На проде после появления Git рекомендуется держать **одну** активную машину редактирования через репо + **`drift-check`**, чтобы не было «ручных только на сервере» изменений без отражения в истории.
